* Early Init

Configuration that runs before the main init process. Sets up package
management, disables GUI chrome, and tunes garbage collection.

** Package Management

Enable package initialization at startup for faster package loading.

#+BEGIN_SRC elisp
(setq package-enable-at-startup t)
(defvar package-quickstart)
#+END_SRC

** GUI Defaults

Disable all GUI chrome before the first frame is created. This prevents
the brief flash of toolbars and scrollbars on startup.

#+BEGIN_SRC elisp
(setq frame-inhibit-implied-resize t)
(menu-bar-mode -1)
(tool-bar-mode -1)
(scroll-bar-mode -1)
#+END_SRC

** Startup Behavior

Suppress the splash screen, startup buffer menu, and file dialog.
Keep the dialog box enabled for interactive prompts.

#+BEGIN_SRC elisp
(setq inhibit-splash-screen t)
(setq use-dialog-box t)
(setq use-file-dialog nil)
(setq inhibit-startup-screen t)
(setq inhibit-startup-buffer-menu t)
#+END_SRC

** Performance

Maximize the GC threshold during startup to reduce collection pauses,
then dial it back after Emacs finishes loading. Also increase the
process output buffer to 1MB for better subprocess performance.

#+BEGIN_SRC elisp
(setq gc-cons-threshold most-positive-fixnum)
(setq read-process-output-max (* 1024 1024))
(setq default-frame-alist '((fullscreen . maximized)))

(add-hook 'emacs-startup-hook
          (lambda ()
            (setq gc-cons-threshold 100000000)))
#+END_SRC
