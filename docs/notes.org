* Notes Module (~config/notes.el~)

Note-taking with org-mode and denote.

** Org Mode

- TODO keywords: =TODO= → =DOING= → =WAITING= → =DONE=
- Agenda file: =~/.notes/20250523T112215--tasks__work.org=
- Remote images downloaded for inline display

| Key     | Command         |
|---------+-----------------|
| =C-c a= | org-agenda      |
| =C-c c= | org-capture     |
| =C-c l= | org-store-link  |
| =C-c C-;= | org-todo      |

** Denote

File-based note-taking. Notes stored in =~/.notes/= with sorted keywords.

| Key         | Command                 |
|-------------+-------------------------|
| =C-c n n=   | denote (create)         |
| =C-c n o=   | denote-open-or-create   |

Predefined keywords: =work=, =personal=, =finance=, =urgent=.

=denote-dired-mode= integrated into Dired for note metadata display.

** Org Download

Enables drag-and-drop and clipboard paste of images into org buffers.
