* Rice Module (~config/rice.el~)

Visual configuration: theme, fonts, modeline.

** Theme

Modus themes with custom styling:

- =modus-vivendi-tinted= loaded by default
- Toggle between =modus-operandi= and =modus-vivendi-tritanopia= via =C-c m m=
- Org blocks get tinted backgrounds
- Headings scale with rainbow colors per level
- Mode line uses accented borderless style

** Fonts

Linux default: =DejavuSansMNerdFontMono= at 105 height, line spacing 1.

Per-OS overrides:
- macOS: =DejavuSansMNerdFontMono= at 140 height (in =mac.el=)
- Windows: =CommitMono= at 130 height, line spacing 3 (in =windows.el=)

** Packages

- =doom-modeline= — rich status bar
- =all-the-icons= — icon support (graphical frames only)
- =diminish= — hides =auto-revert=, =eldoc=, =which-key= from modeline
- =rainbow-delimiters= — color-coded parens in =prog-mode=
- =marginalia= — rich minibuffer annotations (=M-A= to cycle)
- =display-time= — shows clock in HH:MM format
