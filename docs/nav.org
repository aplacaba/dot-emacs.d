* Navigation Module (~config/nav.el~)

Movement and search tools.

** Packages

- =pixel-scroll-precision-mode= — smooth scrolling
- =rg= — ripgrep integration with default bindings enabled
- =avy= — jump to visible text

| Key     | Command           |
|---------+-------------------|
| =C-:=   | avy-goto-char     |
| =M-g e= | avy-goto-word-0   |
| =C-c C-j= | avy-resume      |

- =switch-window= — visual window selector with QWERTY shortcuts and minibuffer input. Overrides =C-x o=.
- =vundo= — visual undo tree (also bound to =C-c v= in bindings.el)
