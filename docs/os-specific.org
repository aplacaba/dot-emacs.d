* OS-Specific Config

** macOS (~config/mac.el~)

- Command key =Meta=, Option key unmapped (=mac-command-modifier 'meta=)
- Font: =DejavuSansMNerdFontMono= at height 140, line spacing 1
- =ultra-scroll= for native smooth scrolling (macOS only)

** Windows (~config/windows.el~)

- =wsl-copy= — copies region to Windows clipboard via =clip.exe=
- Font: =CommitMono= at height 130, line spacing 3
- TRAMP uses =plink= method, auto-save to =c:\Users\A\tmp=
- ispell set to =aspell=
- Loads Roswell helper from =C:/Users/A/.roswell/helper.el=
