* Common Lisp Module (~config/llisp.el~)

SLIME setup for SBCL.

** Configuration

- Expects SLIME installed at =~/.slime= (added to =load-path=)
- Uses =slime-autoloads= for deferred loading
- =inferior-lisp-program= set to =sbcl=
- =g-slime-start= function starts SLIME with prefix arg (used for choosing implementation)

** Requirements

- [[https://github.com/roswell/roswell][Roswell]] or direct SBCL installation
