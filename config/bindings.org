* Bindings

Global keybindings. Loaded last so all packages are available.

** Colemak Layout

The core Colemak adaptation: =C-t= is translated to =C-x= via
=key-translation-map=. All =C-x= commands are reached via =C-t=
instead.

#+BEGIN_SRC elisp
(define-key key-translation-map [?\C-t] [?\C-x])
#+END_SRC

** Window Splits

Splits that automatically move focus to the new window.

#+BEGIN_SRC elisp
(defun vertical-split ()
  (interactive)
  (split-window-vertically) (other-window 1))

(defun horizontal-split ()
  (interactive)
  (split-window-horizontally) (other-window 1))

(global-set-key "\C-x2" #'vertical-split)
(global-set-key "\C-x3" #'horizontal-split)
(global-set-key (kbd "M-o") 'other-window)
#+END_SRC
