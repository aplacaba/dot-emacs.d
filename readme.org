* Emacs Config
** Requirements

Tested on the following operating systems:
- Fedora 42
- MacOS 15
- Windows 11

To install please ensure the following are installed (linux / mac only):

vterm
#+BEGIN_SRC sh
sudo dnf install libvterm
#+END_SRC

sbcl
#+BEGIN_SRC sh
  brew install roswell
#+END_SRC

ripgrep
#+BEGIN_SRC rg
  sudo dnf install ripgrep
#+END_SRC
