* Emacs Config

A modular Emacs configuration targeting Fedora Linux (primary), macOS 15, and Windows 11.

** Features

- Completion stack: Vertico + Corfu + Orderless + Consult + Cape + Tempel
- Git: Magit with transient level 7
- Terminal: vterm + vterm-toggle
- Navigation: Avy, switch-window, ripgrep, consult-line
- Programming: Tree-sitter, flycheck, yasnippet, LSP-ready
- Languages: TypeScript/JSX, Clojure (CIDER), Ruby, Elixir, Python, Common Lisp (SLIME), Nix, YAML, Terraform, Docker, CloudFormation
- Notes: org-mode + denote
- AI: gptel (Gemini)
- Theme: Modus themes with doom-modeline

** Requirements

*** Linux (Fedora)

#+BEGIN_SRC sh
sudo dnf install libvterm ripgrep
#+END_SRC

*** macOS

#+BEGIN_SRC sh
brew install libvterm ripgrep roswell
#+END_SRC

*** Windows

- Install [[https://github.com/akermu/emacs-libvterm][libvterm]] or use eshell (vterm skipped on Windows automatically)

*** Optional

- =sbcl= — Common Lisp REPL via SLIME
- =cfn-lint= (~pip install cfn-lint~) — CloudFormation linting
- =multimarkdown= — Markdown preview
- Gemini API key in =~/.authinfo= or =GEMINI_KEY= env var for gptel

** Installation

#+BEGIN_SRC sh
git clone <repo-url> ~/.emacs.d
#+END_SRC

First launch will install all packages automatically via =use-package= =:ensure=.

After first launch, install tree-sitter grammars by evaluating:

#+BEGIN_SRC elisp
(mapc #'treesit-install-language-grammar (mapcar #'car treesit-language-source-alist))
#+END_SRC

** Architecture

All configuration lives in =config/= loaded as modules via =require= from =init.el=.

#+BEGIN_SRC
~/.emacs.d/
├── early-init.el          # pre-frame setup (GC tuning, GUI defaults)
├── init.el                # package sources, load-path, requires
├── config/
│   ├── core.el            # built-in settings, completion stack, magit, vterm
│   ├── rice.el            # themes, fonts, modeline
│   ├── nav.el             # avy, switch-window, ripgrep
│   ├── notes.el           # org-mode, denote
│   ├── devops.el          # terraform, docker, cfn, pg/pgmacs
│   ├── programming.el     # languages, tree-sitter, flycheck, yasnippet, gptel
│   ├── llisp.el           # SLIME/SBCL
│   ├── bindings.el        # global keybindings (Colemak-aware)
│   ├── mac.el             # macOS overrides
│   └── windows.el         # Windows overrides
├── snippets/              # yasnippet snippets
├── custom.el              # customize settings (gitignored)
└── docs/                  # detailed module documentation
#+END_SRC

Each module follows the same structure:

#+BEGIN_SRC elisp
;;; -*- lexical-binding: t; -*- name.el --- Description
;;; Commentary:
;;; Code:

;; ... use-package forms and config ...

(provide 'name)
;;; name.el ends here
#+END_SRC

** Customizations

OS-specific config is in =config/mac.el= and =config/windows.el=. Emacs =customize= settings are stored in =custom.el= (gitignored).
